Full Stack Web Application

Click here to see a Web Application that I built by myself in one week for Geography 576 at UW-Madison: Geo-spatial Web and Mobile Programming. The intent is to create a safe-walking app for people to report and see patterns in potentially dangerous crimes. There is a version of this app that connects to a real database, however anything you report here will be erased when you refresh, so play around! The design of the app was to be on a mobile device, considering the user group, so view on a phone or small browser for optimal UI. Due to the shear volume of the database, this page may take a while to load.
