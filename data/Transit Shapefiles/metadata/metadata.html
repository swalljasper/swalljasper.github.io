Section 2 Data Quality
Attribute Accuracy Attributes randomly checked by Metro Transit staff. Station names are generally reflective of the branded names of the stations.
Logical Consistency The dataset is topologically 'clean'.
Completeness
Horizontal Positional Accuracy Station locations were created using heads-up digitizing while referencing aerial photography.
Lineage The geography for this dataset was created referencing aerial photography and estimating the centerpoint of each station.

Section 5 Attributes
Overview POINT ATTRIBUTES:

STATION: The name of the transitway station.

X: The longitude in decimal degrees.

Y: The latitude in decimal degrees.

Name: Name of the route which is served by the station

Type: Type of route served by station

Transitway: Transitway name

StationID: Unique station identification number
